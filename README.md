# CASTLE HUNTER - By Navalex (alexandre.navaro@epitech.eu)


### GOAL
Click on little and cute knight to earn coins. Each 15 coins earned, The Wizard will spawn !

### Usage
Run Makefile:
`$ make`
And run the game:
`./my_hunter`
